Project Name
BOQ Exporter

Round Number
R09

Revit Add-in Tab
KAITECH-BD-R09

Panel
General

Description
BOQ Exporter is a Revit add-in that extracts quantities and item counts from Revit models and exports them to a professionally formatted Excel BOQ report. It supports category selection, project metadata input, logo embedding, summary tables, and visual analysis charts.

Features

* Category selection through a user interface
* Project metadata input (Project Name, Client Name, Location, Issue Date)
* Company logo embedding in Excel output
* Detailed BOQ report including quantities, unit rates, total cost, and remarks
* BOQ summary report with item usage counts
* Category analysis table with pie chart visualization
* Alternating row shading and subtotal formatting for readability

Setup Instructions

1. Run the setup file located in the Release folder.
2. The installer automatically places the .addin manifest and required DLL files into the correct Revit Addins directories.
3. Supported Revit versions are installed simultaneously.
4. Launch Revit and locate the “KAITECH-BD-R09” tab.
5. Open the “General” panel and click “BOQ Exporter” to start the add-in.

Compatibility

* Revit 2022
* Revit 2023
* Revit 2024
* Revit 2025
* Revit 2026

Usage Instructions

1. Open a Revit project.
2. Launch BOQ Exporter from the KAITECH-BD-R09 tab.
3. Select required categories and enter project metadata.
4. Click “Extract” to preview grouped BOQ items.
5. Click “Export” to generate the Excel BOQ report.
